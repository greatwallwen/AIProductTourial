## 产品问题

电商售后场景中，售后产品经理围绕退换货履约做日常判断。售后产品经理需要围绕“退换货履约”完成“优化跨角色流程和异常处理”，并把结果转成可验证的产品交付物。核心不是把页面做出来，而是把指标、异常、责任和行动连成闭环。

## 岗位与业务对象

- 岗位：售后产品经理
- 业务对象：电商退换货履约
- 行业：电商售后
