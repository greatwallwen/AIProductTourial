# DESIGN · 石墨 HUD（graphite-hud）

> 一套**独立**的深色大屏设计系统。与其它 design/*.md 色板/布局/气质各不相同。适用大屏范式：**上图下表**；SaaS 原型：executive_dashboard、hr_efficiency_screen。

## 视觉主题与气质
石墨黑高端科技、青蓝高光、HUD 网格。整体像可交付的前端大屏截图：深色渐变底 + 玻璃拟态卡片 + 细边框 + 柔和发光 + HUD 网格 + 精密图表线条；中文短标签、真实业务数字、信息密度高、异常高亮。

## 色板与语义（hex）
- 背景：`#0a0e14` → `#0d1220`（近黑渐变）｜网格线 `#16202e`
- 面板/卡片：`#121826`（次级 `#0f1522`）｜边框 `#1e2a3a`
- 文字：主 `#e6edf6`／次 `#9fb0c3`／弱 `#5b6b7f`
- 数据高光：主 `#22d3ee`／辅 `#38bdf8`
- 状态：正常 `#34d399`／预警 `#fbbf24`／严重 `#f87171`
- 发光：`0 0 18px rgba(34,211,238,.35)`

## 字体层级
系统中文无衬线（PingFang SC / Microsoft YaHei）；大数字 tabular-nums；标题 650、KPI 700、标签 500；短标签避免英文乱码。

## 组件
卡片：玻璃拟态（半透明面板 + 1px 边框 + 顶部微高光）；徽章：状态色底 + 深色字；表格：细分隔线、悬停高亮、状态徽章；图表：上图下表 主图 + HUD 网格 + 发光描边。

## 布局与密度
顶部总览指标（KPI 大数字 + 趋势）→ 中部主图（上图下表）→ 底部明细表（排行/告警/时间戳）。16:9、驾驶舱高密度。

## 深度
背景网格（HUD）+ 卡片轻投影 + 数据高光发光；层次靠明度与发光，不靠重阴影。

## 响应
桌面优先（1440 基准）；网格 auto-fill；宽表 overflow-x 自滚，页面不横向滚动。
