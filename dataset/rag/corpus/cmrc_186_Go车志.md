# Go车志

《Go车志 BuyCarTV》，于2007年7月－2009年于台湾交通电视台播放；2009年开始以YouTube为主要平台。节目内容为台湾市场与国外的新车试驾评论以及台湾汽车零配件报导为主。让观众了解台湾汽车市场的生态与变化。创办人及主持人为曾任车讯国际总编辑的「William」朱嘉伟。该节目与Yahoo!奇摩汽车的合作，改变台湾网友上网搜寻车辆资讯的习惯，从文字图片转变成影音为主的汽车影片。为推广推广台湾合法安全赛车运动，于2013年9月成立「Go车志车队」，2015年持续参加OTGP与台湾大赛车等赛车活动。2015参赛车手名单：朱嘉伟、游启明、吕文斌、洪佳进等四位车手，以及一位随机轮替的车手。。2016年参赛车手：朱嘉伟（88号）2017年参赛车手：朱嘉伟（308号）、Joe Kwok（408号）香港车手朱嘉伟 （308号）：参赛车辆(Peugeot RCZ Racing Cup)，2017年TSF闭幕站（Peugeot 308 TCR Racing Cup）Joe Kwok（408号）：参赛车辆(Honda civic Type R )(FD2)
