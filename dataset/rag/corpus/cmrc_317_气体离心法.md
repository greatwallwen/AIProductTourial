# 气体离心法

气体离心法是一种制造浓缩铀的机械式方法，用以从天然铀中将铀-238分离出铀-235。离心机是依靠离心力原理运作，可以加速分子以上大小的物质。当圆筒状物体开始旋转，六氟化铀气体就逐一通过各离心筒，逐渐累积纯化。气体离心法是取代早期气体扩散法的核原料浓缩技术。此法的最大优点是取得浓缩铀-235的过程可以比气体扩散法节省相当多能量。最初在1919年被提出，首次成功于1934年。J.W.Beams和一些助手在维吉尼亚大学研发分开两种氯同位素利用真空高速离心机的方案。最初曼哈顿计划放弃用这种方法研发首颗原子弹，但是后来证明此方法更好。气体离心法利用离心力分离铀的分子，可以用于铀的气体或液体。利用高速强行分离铀-235的原子。铀-238同位素重分子气体比铀-235轻分子气体更容易在圆筒的近壁处得到聚集。在近轴处聚集的气体被导出，并输送到另一台离心机进一步分离。随着气体（液体）穿过一系列离心机，其铀-235同位素分子被逐渐聚集浓度。通常一个气体离心厂需要上千组离心机才能浓缩到所需浓度。气体离心法使用独特设计的离心机使气体或液体能不间断的在各离心筒中流动。不同于其他工业用的离心法需要批量处理，此设备可以连续运转加工「铀流」，也因为要极多个离心筒才能把铀浓缩到武器用程度，铀也具有放射性不同于其他工业产品所以有此设计。一组气体离心机单元组成有：筒状旋转槽，套管，电动马达，三条物质流动用管线。设计中有一全套管套住整台离心机。防止有任何物体流出喷出伤害到工作人员。筒状旋转槽定位在套管正中心，可以隔离任何空气。电动马达持续旋转，施加离心力在筒中的任何物质上。外部有两条输出管线，一条在离心机上部一条在下部。轻分子将从上方管线被抽出；重分子将从下方管线抽出。输出管线可以把浓缩过的铀传送到下一个离心组继续浓缩。离心机单元（SWU）的测量标准质量单位（公制单位公斤-SWU）。必须达成下列公式效果：SWU = WV(xw) + PV (xp) – FV (xf)代数：W = 消耗质量（xw）P = 产出质量（xp）F = 输入质量（xf）V(x) = (1-2x)ln((1-x)/x)
