# 规范来源与检索

本目录规范由**项目实践素材**抽取通用规则（去业务专有），并对齐/检索以下业界主流工程规范（GitHub/网络）：

- Google Style Guides — https://google.github.io/styleguide/
- Software Engineering at Google（swe-book）— https://abseil.io/resources/swe-book/
- Airbnb JavaScript Style Guide — https://github.com/airbnb/javascript
- Conventional Commits（提交规范）— https://www.conventionalcommits.org/
- OWASP Top 10（安全）— https://owasp.org/www-project-top-ten/
- The Twelve-Factor App（配置/部署）— https://12factor.net/
- Effective Go — https://go.dev/doc/effective_go
- awesome-guidelines（编码规范聚合）— https://github.com/Kristories/awesome-guidelines
- programming-best-practices — https://github.com/dereknguyen269/programing-best-practices
- software-development-best-practices — https://github.com/dronezzzko/software-development-best-practices

> 素材原为嵌入式学习平台（STM32）前后端规范，本仓库仅取其**通用工程规则**，业务专有部分不纳入。
